## Survey Providers

### 1. Toluna (Survey Platform)

**Registration URL:** https://corporate.toluna.com/contact-us

**Type:** B2B Partnership (requires business account)

**Requirements:**

- Registered business
- Website/app with privacy policy
- Expected monthly survey completions
- Business email

**API Access:**

1. Contact Toluna's B2B team via their corporate site
2. Request "Publisher API" or "Panel Integration" access
3. Provide your app details and expected user volume
4. Sign partnership agreement

**What You'll Get:**

- API Key
- API Secret
- Publisher ID
- API Documentation URL
- Sandbox credentials for testing

**Commission:** Negotiate (typically 20-30% of survey reward)

**Nigeria Support:** ✅ Yes

**Notes:** Toluna is one of the largest survey platforms. Approval may take 2-4 weeks.

### 2. OpinionWorld (Dynata)

**Registration URL:** https://www.opinionworld.com/en-ng/ (for Nigeria)

**Parent Company:** Dynata - https://www.dynata.com/partners/

**Type:** B2B Partnership

**Requirements:**

- Established platform with active users
- Minimum 1,000 monthly active users
- Business registration documents
- Data privacy compliance (GDPR-like)

**API Access:**

1. Visit https://www.dynata.com/partners/
2. Click "Become a Partner"
3. Select "Sample Integration" or "Panel Partnership"
4. Fill out partnership form
5. Wait for account manager contact

**What You'll Get:**

- API Endpoint URL
- Client ID
- Client Secret
- Integration Guide
- Test environment access

**Commission:** Negotiate (typically 15-25%)

**Nigeria Support:** ✅ Yes

**Notes:** Dynata owns multiple survey brands. One partnership can give access to multiple survey sources.

### 3. Surveytime

**Registration URL:** https://surveytime.io/affiliates

**Type:** Affiliate Program (easier to join)

**Requirements:**

- Website or mobile app
- Traffic source description
- Payment method (PayPal or bank)

**API Access:**

1. Visit https://surveytime.io/affiliates
2. Sign up as an affiliate
3. Request API access via support ticket
4. Provide technical documentation of your integration

**What You'll Get:**

- Affiliate ID

- Dashboard access

**Commission:** Fixed $1 per completed survey (you decide user payout)

**Nigeria Support:** ⚠️ Limited (check current availability)

**Notes:** Easier to join than others. May start with tracking links before API access.

## Social Media Task Providers

### 4. Picoworkers

**Registration URL:** https://picoworkers.com/

**Type:** Open Platform (easier access)

**Requirements:**

- Email verification
- Phone verification
- Business account upgrade (for API)

**API Access:**

1. Create account at https://picoworkers.com/
2. Verify email and phone
3. Contact support: support@picoworkers.com
4. Request "Employer API" access
5. Provide app details and use case

**What You'll Get:**

- API Key
- API Documentation
- Webhook URLs for callbacks
- Dashboard access

**Commission:** Platform takes 20%, you can add your markup

**Nigeria Support:** ✅ Yes

**Notes:** One of the easiest to integrate. Good starting point.

---

### 5. Microworkers

**Registration URL:** https://www.microworkers.com/

**Type:** Open Platform

**Requirements:**

- Email verification
- ID verification (for withdrawals)
- Business account (for API)

**API Access:**

1. Register at https://www.microworkers.com/
2. Upgrade to "Employer" account
3. Email api@microworkers.com requesting API access
4. Provide:
   - Business details
   - Integration plan
   - Expected task volume

**What You'll Get:**

- API Key
- API Secret
- REST API documentation
- Sandbox environment

**Commission:** 15% platform fee + your markup

**Nigeria Support:** ✅ Yes

**Notes:** Similar to Picoworkers. Good task variety.

## Micro-Task Providers

### 13. Amazon Mechanical Turk (MTurk)

**Registration URL:** https://requester.mturk.com/

**Type:** AWS Service (requires AWS account)

**Requirements:**

- AWS account
- Business verification
- Credit card for AWS billing
- Tax information (W-9 or W-8)

**API Access:**

1. Create AWS account at https://aws.amazon.com/
2. Navigate to MTurk Requester: https://requester.mturk.com/
3. Complete requester registration
4. Link payment method
5. Create IAM credentials for API access

**What You'll Get:**

- AWS Access Key ID
- AWS Secret Access Key
- MTurk API endpoint
- Sandbox environment

**Commission:** You set task prices + 20-40% Amazon fee

**Nigeria Support:** ⚠️ Limited (requires US/international payment method)

**Notes:** Most established platform but has geographic restrictions.

## Important Notes

### Starting Strategy

**Phase 1: Easy Wins (Start Here)**

1. **Hideout.tv** - Easiest video provider
2. **Picoworkers** - Easy social media tasks
3. **Lootably** - Multi-type content

**Phase 2: Expand** 4. **Surveytime** - Affiliate program 5. **Microworkers** - More social tasks 6. **AdGem** - Additional videos

**Phase 3: Scale** 7. **Toluna/Dynata** - Premium surveys 8. **Mistplay** - Game rewards 9. **Clickworker** - Micro-tasks

### General Tips

1. **Start Small:** Begin with 2-3 providers that are easiest to join
2. **Test First:** Use sandbox/test environments before going live
3. **Read Terms:** Understand payout terms, minimums, and schedules
4. **Compliance:** Ensure your privacy policy covers data sharing
5. **User Experience:** Don't overwhelm users with too many task types initially

### Payment Considerations

- Most providers pay NET-30 or NET-60
- Minimum payout thresholds vary ($50-$500)
- Payment methods: PayPal, Wire Transfer, Payoneer
- Consider cash flow when setting user payouts

### Nigeria-Specific Considerations

**Best Providers for Nigeria:**

- ✅ Hideout.tv
- ✅ Picoworkers
- ✅ Microworkers
- ✅ Lootably
- ✅ RapidWorkers

**Limited/Check First:**

- ⚠️ Mistplay
- ⚠️ Surveytime
- ⚠️ MTurk

```

Hello [Provider Name] Team,

I'm reaching out from Earn9ja, a task-based earning platform operating in Nigeria and West Africa.

We're interested in integrating [Provider Name] tasks into our mobile application to provide our users with diverse earning opportunities.

Platform Details:
- Platform: Mobile app (React Native - iOS & Android)
- Current Users: [Your number] monthly active users
- Target Market: Nigeria, Ghana, Kenya
- Website: [Your website]

We're looking to integrate via API to offer your [surveys/tasks/videos] to our user base.

Could you please provide information about:
1. API access requirements
2. Partnership terms and commission structure
3. Technical documentation
4. Approval timeline

Thank you for your consideration.

Best regards,
[Your Name]
[Your Title]
[Contact Information]
```

### Next Steps

1. **Choose 2-3 providers** from Phase 1 (easy wins)
2. **Register accounts** and apply for API access
3. **While waiting for approval**, build the core infrastructure (Tasks 1-7 in implementation plan)
4. **Test with sandbox** credentials once received
5. **Go live** with one provider first
6. **Monitor performance** before adding more
