# Real Estate Platform

## Project Overview

A comprehensive real estate listing platform built with Next.js 15 where users can browse, search, and filter properties. Agents can list properties, manage inquiries, and track leads. Features include advanced search with map integration, virtual tours, appointment scheduling, and mortgage calculators.

## Key Features

- Property listings with detailed information
- Advanced search and filtering
- Interactive map integration (Google Maps/Mapbox)
- Image galleries and virtual tours
- User authentication (buyers, sellers, agents)
- Agent profiles and property management
- Inquiry and appointment system
- Favorites/saved properties
- Mortgage calculator
- Property comparison
- Neighborhood information
- Admin dashboard

## Tech Stack

- **Frontend**: Next.js 15, TypeScript, Zustand, @tanstack/react-query
- **State Management**: Zustand for client state, React Query for server state
- **Backend**: Next.js API Routes, MongoDB Atlas, Mongoose
- **Maps**: Google Maps API or Mapbox
- **File Storage**: Cloudinary
- **Authentication**: JWT, bcrypt
- **Styling**: Tailwind CSS v4, DaisyUI
- **Icons**: Lucide React
- **Database**: MongoDB Atlas (cloud database)
