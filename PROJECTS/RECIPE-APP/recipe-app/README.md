# Recipe Sharing App

## Project Overview
A social recipe sharing platform where users can create, share, and discover recipes. Features include ingredient lists, step-by-step instructions, ratings, reviews, and meal planning capabilities.

## Key Features
- Recipe CRUD with ingredients and instructions
- Image upload for recipe photos
- Search and filter by cuisine, diet, difficulty
- Rating and review system
- Save favorite recipes
- Meal planning calendar
- Shopping list generation
- Nutritional information
- User profiles and following system
- Recipe collections/cookbooks

## Tech Stack
- **Frontend**: React, React Router, Redux Toolkit, React Hook Form
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **File Storage**: Multer, Cloudinary
- **Authentication**: JWT, bcrypt
- **Styling**: Tailwind CSS or Material-UI

## Learning Outcomes
- Complex form handling with dynamic fields
- Advanced filtering and search
- Social features (following, favorites)
- Calendar integration
- Data aggregation (shopping lists)
- Image optimization
- User-generated content management
