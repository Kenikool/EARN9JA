## Tech Stack
- **Frontend**: React 19, TypeScript, Vite, TailwindCSS, DaisyUI
- **State Management**: React Query + Zustand (no Redux)
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **Real-Time**: Socket.io (live chat, notifications)
- **Payment**: Stripe, Flutterwave, Paystack
- **File Storage**: Cloudinary
- **Authentication**: JWT, bcrypt, 2FA
- **PWA**: Service Workers, Web Push
- **i18n**: react-i18next
- **AI/ML**: TensorFlow.js (optional)

## Getting Started
```bash
# Clone the repository
git clone <your-repo-url>

# Install dependencies
cd server && npm install
cd ../client && npm install

# Set up environment variables
cp server/.env.example server/.env
cp client/.env.example client/.env

# Start development servers
cd server && npm run dev
cd client && npm run dev
```
